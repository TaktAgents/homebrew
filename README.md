# homebrew-takt

Официальный Homebrew Tap для установки утилит **Takt**.

## Установка

### 1. Подключение репозитория tap
```bash
brew tap takt-agents/takt
```

### 2. Установка GUI-приложения (MenuBarExtra)
```bash
brew install --cask takt
```

### 3. Установка CLI-клиента
```bash
brew install takt
```

## Обновление

Для обновления формул и касков выполните:
```bash
brew update
brew upgrade takt
brew upgrade --cask takt
```
